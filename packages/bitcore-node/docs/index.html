bitcore-node

Bitcore Node

Requirements:

  • Trusted P2P Peer
  • MongoDB Server >= v3.4

Checkout the repo

git clone git@github.com:bitpay/bitcore.git
git checkout master
npm install

Set up your bitcore.config.json file in ./bitcore
